**Path:** `infraestructura/aws/`
**Belongs to:** [`infraestructura/`](../README.en.md)

## What it is and why it exists

This folder is **intentionally empty** in the project's current state. It's reserved as
the future location for infrastructure code (Terraform/CDK) that would implement the AWS
architecture design documented in Phase 8, should the project move from a reference design
to a real deployment.

## Contents of this folder

Currently no files. The complete architecture design (no real deployment) is documented
in [`docs/aws/`](../../docs/aws/INDEX.en.md).

## Relationship to the methodology

The thesis proposal's Phase 8 scope is a **documented reference design**, not a real AWS
deployment (see [`docs/aws/INDEX.md`](../../docs/aws/INDEX.en.md) §Scope). This folder
marks where the IaC would live if that scope were expanded in future work — it keeps the
repository structure consistent with the architectural design without anticipating code
outside the current scope.
